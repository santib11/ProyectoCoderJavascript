Tienda informatica

Este simulador es una herramienta diseñada para que un administrador pueda gestionar un inventario de productos de manera sencilla. Permite registrar usuarios e iniciar sesion ademas de permitir agregar nuevos productos, ver la lista actual de productos y eliminar productos.

Todo el flujo del simulador se lleva a cabo a través de un menú interactivo que utiliza cuadros de diálogo como prompt, alert y confirm ademas de estructuras condicionales, bucles, funciones y arrays.
